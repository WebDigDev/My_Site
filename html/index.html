Профессиональная верстка сайтов

Профессиональная верстска сайтов любой сложности. Вы получите качественно сверстанный, современный, динамический сайт. Работаю с последними технологиями и по последним стандартам- HTML 5, CSS 3, Java Script, jQuery. От вас потребуется макет сайта в PSD (PhotoShop Document) или в AI (Adobe Illustrator) и техническое задание(ТЗ), чем подробнее ТЗ тем лучше будут поняты ваши желания. Задание всегда можно будет обсудить подробнее удобным для вас способом. Цена данной услуги зависит от объема работы, ваших требований и сложности.

Этот абзац для тех, кто не до конца понимает, что такое верстка. Вёрстка веб-страниц — создание структуры html-кода, размещающего элементы веб-страницы (изображения, текст и т. д.) в окне браузера, согласно разработанному макету, таким образом, чтобы элементы дизайна выглядели аналогично макету.

  • Изменение разметки HTML
  • Изменение таблиц стилей CSS
  • Добавление интереактивности Java Script, jQuery
500 - 5000 руб.

Цена данной услуги будет зависеть от колличества правок и от их сложности.
